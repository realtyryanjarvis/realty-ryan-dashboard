# Alt3 Smart Homes — Visual Style Guide for Social Media

---

## Brand Foundation

- **Font:** Neue Montreal (Bold, Regular, Light)
- **Colors:**
  - Navy: #113665 (primary — backgrounds, headlines)
  - Warm Taupe: #BEB1A2 (accents, secondary text, dividers)
  - Light Cream: #E1D8CE (soft backgrounds, text on dark)
  - White: #FFFFFF (clean backgrounds, text)
  - Black: #000000 (editorial contrast, cinematic)
- **Tagline:** "Simple. Seamless. Smart — by design."
- **Tone:** Luxury, minimal, editorial. Architectural Digest meets technology.

---

## Photography & Video Guidelines

### Aesthetic
- **Mood:** Warm, aspirational, lived-in luxury. Homes should look like someone beautiful lives there — not a showroom.
- **Light:** Always natural light preferred. Golden hour is king. Interior shots: show the LIGHTING DESIGN in action (warm Ketra glow, layered scenes). Avoid flat overhead fluorescent look.
- **Color grading:** Warm tones, slight desaturation. Not orange. Think: creamy highlights, rich shadows, warm midtones. Reference: Architectural Digest, Cereal Magazine, Kinfolk.

### Camera Angles
- **Wide establishing shots** — show the full room, let the architecture breathe
- **Low angle** — makes spaces feel grand
- **Slow gimbal movements** — cinematic walkthrough feel for video
- **Detail shots** — close-ups on keypads, speakers flush in ceiling, shading fabric texture, light hitting surfaces
- **POV shots** — for Reels, use first-person perspective entering rooms

### What to Avoid
- ❌ Fish-eye or ultra-wide distortion
- ❌ Flash photography (kills the lighting design)
- ❌ Cluttered frames — if it's messy, don't shoot it
- ❌ Screen-heavy shots (avoid showing too many iPads/phones — show the RESULT, not the device)
- ❌ Stock photos that look like stock photos
- ❌ Overly saturated or HDR looks
- ❌ Tech-bro aesthetic (no RGB, no gaming vibes, no "look at my setup")

### Video Specs
- **Reels:** 9:16, 1080x1920px, 30-60 seconds preferred
- **Feed video:** 1:1 or 4:5
- **Stories:** 9:16, 1080x1920px
- **Frame rate:** 24fps for cinematic, 30fps for clean, 60fps for slow-mo details
- **Music:** Licensed or royalty-free. Ambient, electronic, acoustic. Never generic "upbeat corporate."

## Template Layouts

### Single Image Post (1080x1350 portrait preferred)
```
┌──────────────────────┐
│                      │
│    [PHOTO - 70%]     │
│                      │
├──────────────────────┤
│  Navy or Cream bar   │
│  Caption/quote text  │
│  Logo bottom-right   │
└──────────────────────┘
```
- Photo takes up majority. Text overlay only if needed — keep it minimal.
- If using text overlay on photo: navy semi-transparent bar with cream text.

### Carousel Slide (1080x1350)
```
COVER SLIDE:
┌──────────────────────┐
│                      │
│   [Large Headline]   │
│   [Subtext]          │
│                      │
│          → Swipe     │
│              Logo    │
└──────────────────────┘

CONTENT SLIDE:
┌──────────────────────┐
│  [Number/Icon]       │
│                      │
│  Headline            │
│  Body text           │
│                      │
│  ─── accent line ─── │
│              Logo    │
└──────────────────────┘
```
- Alternate between navy background/cream text and white background/navy text
- One idea per slide maximum
- Generous margins (80px minimum on all sides)

### Story Template (1080x1920)
```
┌──────────────────────┐
│                      │
│                      │
│   [Center-aligned    │
│    bold statement]   │
│                      │
│   [Smaller detail    │
│    text below]       │
│                      │
│                      │
│              Logo    │
└──────────────────────┘
```
- Full-color backgrounds (navy, cream, or photo)
- Text centered vertically
- Interactive stickers (polls, questions, sliders) below center

## Font Usage on Graphics

| Element | Weight | Size (relative) | Color |
|---------|--------|-----------------|-------|
| Main headline | Bold | 48-64pt | Cream on dark / Navy on light |
| Subheadline | Regular | 28-36pt | White on dark / Black on light |
| Body text | Light or Regular | 20-26pt | White or Navy |
| Caption/CTA | Light | 16-20pt | Taupe #BEB1A2 |
| Numbers (big) | Bold | 120-180pt | Taupe (watermark-style) |

- **Never use more than 2 weights on one slide**
- **Line spacing:** 1.3–1.5x for readability
- **Letter spacing:** Slight tracking on headlines (+20 to +50)
- **Fallback font:** If Neue Montreal unavailable, use Inter or Helvetica Neue

## Color Overlay Treatments

### Photo Overlay — Dark
- Navy #113665 at 60-70% opacity over photo
- Cream #E1D8CE text
- Use for: quotes, stats, dramatic statements

### Photo Overlay — Light
- White #FFFFFF at 50-60% opacity
- Navy #113665 text
- Use for: subtle branded look, softer mood

### Gradient
- Navy #113665 → Black #000000 (bottom to top)
- Use for: cinematic Reel covers, evening scenes

### Solid Backgrounds
- Navy: for authority, education, data
- Cream: for warmth, approachability, lifestyle
- White: for clean, modern, technical
- Black: for premium, editorial, high-contrast (use sparingly)

---

## Logo Placement Rules

- **Position:** Bottom-right corner (default) or bottom-center (carousels, brand-focused)
- **Size:** Small and subtle. Never the focal point. Max 8% of frame width.
- **Versions:**
  - White logo on dark backgrounds
  - Navy logo on light backgrounds
  - Never use the logo on a busy/detailed part of a photo
- **Clear space:** Minimum 40px padding around logo on all sides
- **Never:** stretch, rotate, add effects, or place on clashing backgrounds

## Do's and Don'ts

### ✅ DO
- **Show the RESULT** of technology — warm rooms, beautiful light, peaceful spaces
- **Use negative space** — let the design breathe
- **Feature real projects** — authentic > polished stock
- **Maintain consistency** — same color palette, same font, same mood across all posts
- **Show people in spaces** — lived-in luxury, not empty showrooms (when possible)
- **Use before/after** — powerful storytelling tool for lighting and shading
- **Keep text minimal** on photos — the image should speak
- **End every carousel with a CTA** — website, DM, or save prompt

### ❌ DON'T
- **Show messy wires or ugly install phases** (unless it's a satisfying transformation)
- **Use rainbow colors or neon** — this is not a gaming channel
- **Overload slides with text** — if it needs a paragraph, use the caption
- **Show competitor products** or name them negatively
- **Use memes or trends** that feel cheap (selective trending audio is fine if it fits)
- **Post blurry or poorly lit photos** — if the lighting looks bad, the smart home looks bad
- **Use more than 3 colors per graphic** — keep it tight
- **Post without the Alt3 logo** — subtle brand presence on everything
- **Feature only empty rooms** — lifestyle > architecture alone
- **Use Comic Sans, Papyrus, or any font that isn't Neue Montreal** on branded graphics
